# Desenvolvedor

## Desenvolver este pacote para Atom

Sinta-se à vontade para enviar pull requests, reportar issues ou solicitar novos recursos!

Para modificar e desenvolver o pacote **markdown-preview-enhanced**, você precisa realizar a [instalação local](installation.md?id=install-from-github).

Após concluir a instalação do pacote, siga estes passos:

- Abra a pasta **markdown-preview-enhanced** no **Atom Editor** em **View->Developer->Open in Dev Mode...**
- Então você pode modificar o código.
  Toda vez que atualizar o código, você precisa pressionar <kbd>cmd-shift-p</kbd> e escolher `Window: Reload` para recarregar o pacote e ver as alterações.

> Como a versão Atom é escrita em TypeScript, recomenda-se instalar `atom-typescript` para ajudar no desenvolvimento do pacote.  
> Bem... Eu desenvolvi a versão atom usando o vscode.

## Desenvolver este pacote para VS Code

Clone [vscode-markdown-preview-enhanced](https://github.com/shd101wyy/vscode-markdown-preview-enhanced), depois `yarn install`, abra no **vscode** e comece a depurar.
