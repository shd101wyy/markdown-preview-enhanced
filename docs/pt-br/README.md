# Markdown Preview Enhanced

**Markdown Preview Enhanced** é uma extensão Markdown **SUPER PODEROSA** para o [**Visual Studio Code**](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced).
O objetivo deste projeto é oferecer uma experiência maravilhosa de escrita em Markdown.

Publique [aqui](https://github.com/shd101wyy/vscode-markdown-preview-enhanced/issues) se você quiser solicitar novos recursos ou reportar bugs ;)
