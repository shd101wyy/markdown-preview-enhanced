# Auxiliar de Imagem

O auxiliar de imagem ajuda você a copiar e fazer upload de imagens rapidamente.

## Uso

Para abrir o auxiliar de imagem, você pode executar o comando `Markdown Preview Enhanced: Image Helper`, ou clicar com o botão direito na visualização e escolher a opção `Image Helper`.

![Screen Shot 2017-08-06 at 10.06.48 PM](https://i.loli.net/2017/08/07/5987d95bae68b.png)

## Upload rápido de imagem (apenas Atom)

Você pode fazer upload rápido de uma imagem simplesmente arrastando o arquivo de imagem para o editor Markdown (não para a visualização).

![upload](https://i.loli.net/2017/08/07/5987db34cb33c.gif)
