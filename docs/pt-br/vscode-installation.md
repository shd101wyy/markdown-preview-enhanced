# Instalação no VS Code

Há várias formas de instalar este pacote.

## Instalar pelo VS Code (Recomendado)

Abra o editor **vscode**, pesquise `markdown-preview-enhanced` em `Extensions`, depois clique no botão `Install`.

Você também pode instalar esta extensão no VS Code para Web. Basta abrir o [VS Code for the web](https://vscode.dev/), pesquisar `markdown-preview-enhanced` em `Extensions` e clicar no botão `Install`.

![screen shot 2017-07-14 at 12 23 32 am](https://user-images.githubusercontent.com/1908863/28199365-bb03a570-682a-11e7-8f65-d7d2b258d583.png)

## Instalar pelo GitHub

- Baixe o arquivo `markdown-preview-enhanced-*.vsix` em [Releases](https://github.com/shd101wyy/vscode-markdown-preview-enhanced/releases).
- Abra o **vscode**, execute `Extension: Install from VSIX`, depois selecione o arquivo `*.vsix` que você baixou.
