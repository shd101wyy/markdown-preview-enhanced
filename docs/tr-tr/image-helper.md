# Görsel Yardımcısı

Görsel yardımcısı, görsellerinizi hızlıca kopyalamanıza ve yüklemenize yardımcı olur.

## Kullanım

Görsel yardımcısını açmak için `Markdown Preview Enhanced: Image Helper` komutunu çalıştırabilir veya önizlemede sağ tıklayıp `Image Helper` seçeneğini seçebilirsiniz.

![Screen Shot 2017-08-06 at 10.06.48 PM](https://i.loli.net/2017/08/07/5987d95bae68b.png)

## Hızlı görsel yükleme (Yalnızca Atom)

Görsel dosyanızı markdown editörünüze (önizlemeye değil) sürükleyip bırakarak hızlıca görsel yükleyebilirsiniz.
