# VS Code'a Kurulum

Bu paketi kurmanın birkaç yolu vardır.

## VS Code'dan kurulum (Önerilen)

**vscode** editörünü açın, ardından `Extensions` bölümünde `markdown-preview-enhanced` arayın ve `Install` düğmesine tıklayın.

Bu uzantıyı Web için VS Code'a da kurabilirsiniz. Sadece [Web için VS Code](https://vscode.dev/)'u açın, ardından `Extensions` bölümünde `markdown-preview-enhanced` arayın ve `Install` düğmesine tıklayın.

## GitHub'dan kurulum

- [Releases](https://github.com/shd101wyy/vscode-markdown-preview-enhanced/releases) sayfasından `markdown-preview-enhanced-*.vsix` dosyasını indirin.
- **vscode**'u açın, `Extension: Install from VSIX` komutunu çalıştırın, ardından yeni indirdiğiniz `*.vsix` dosyasını seçin.
