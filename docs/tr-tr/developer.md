# Geliştirici

## Bu Paketi Atom için Geliştirme

Pull request göndermekten, sorun bildirmekten veya yeni özellik istemekten çekinmeyin!

**markdown-preview-enhanced** paketini değiştirmek ve geliştirmek için [yerel kurulum](installation.md?id=install-from-github) yapmanız gerekir.

Paketi kurduktan sonra şu adımları izleyin:

- **markdown-preview-enhanced** klasörünü **Atom Editörü**'nde **View->Developer->Open in Dev Mode...** menüsünden açın.
- Ardından kodu değiştirebilirsiniz.
  Kodu her güncelledikten sonra, paketi yeniden yükleyerek değişiklikleri görmek için <kbd>cmd-shift-p</kbd> tuşlarına basıp `Window: Reload` seçeneğini seçin.

> Atom sürümü TypeScript ile yazıldığından, paketi geliştirmeye yardımcı olması için `atom-typescript` kurulması önerilir.  
> Evet... atom sürümünü vscode kullanarak geliştirdim.

## Bu Paketi VS Code için Geliştirme

[vscode-markdown-preview-enhanced](https://github.com/shd101wyy/vscode-markdown-preview-enhanced)'i klonlayın, ardından `yarn install` komutunu çalıştırın, **vscode**'da açın ve hata ayıklamaya başlayın.
