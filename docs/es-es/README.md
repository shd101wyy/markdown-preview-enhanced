# Markdown Preview Enhanced

**Markdown Preview Enhanced** es una extensión de Markdown **MUY PODEROSA** para [**Visual Studio Code**](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced).
El objetivo de este proyecto es brindarte una maravillosa experiencia de escritura en Markdown.

Publica [aquí](https://github.com/shd101wyy/vscode-markdown-preview-enhanced/issues) si solicitas nuevas funcionalidades o deseas reportar errores ;)
