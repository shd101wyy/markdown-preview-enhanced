# Usos

## Comandos

Puedes presionar <kbd>cmd-shift-p</kbd> en el editor Atom para abrir la <strong>Paleta de Comandos</strong>.

> La tecla <kbd>cmd</kbd> en _Windows_ es <kbd>ctrl</kbd>.

_Fuente Markdown_

- <strong>Markdown Preview Enhanced: Toggle</strong>  
  <kbd>ctrl-shift-m</kbd>  
  Activar/desactivar la vista previa del archivo Markdown.

- <strong>Markdown Preview Enhanced: Toggle Zen Mode </strong>  
  Activar/desactivar el modo de escritura sin distracciones.

- <strong>Markdown Preview Enhanced: Customize Css</strong>  
  Personalizar el CSS de la página de vista previa.  
  Aquí tienes un [tutorial](customize-css.md) rápido.

- <strong>Markdown Preview Enhanced: Create Toc </strong>  
  Generar TOC (requiere que la vista previa esté activa). [La documentación está aquí](toc.md).

- <strong>Markdown Preview Enhanced: Toggle Scroll Sync </strong>  
  Activar/desactivar la sincronización del scroll en la vista previa.

- <strong>Markdown Preview Enhanced: Sync Source </strong>  
  <kbd>ctrl-shift-s</kbd>  
  Desplazar la vista previa para coincidir con la posición del cursor en la fuente Markdown.

- <strong>Markdown Preview Enhanced: Toggle Live Update </strong>  
   Activar/desactivar la actualización en tiempo real de la vista previa.  
   Si está desactivado, la vista previa solo se renderizará cuando se guarde el archivo.

- <strong>Markdown Preview Enhanced: Toggle Break On Single Newline </strong>  
  Activar/desactivar el salto de línea en una sola nueva línea.

- <strong>Markdown Preview Enhanced: Insert New Slide </strong>  
  Insertar una nueva diapositiva y entrar al [modo presentación](presentation.md)

- <strong>Markdown Preview Enhanced: Insert Table </strong>  
  Insertar una tabla Markdown.

- <strong>Markdown Preview Enhanced: Insert Page Break </strong>  
  Insertar un salto de página.

- <strong> Markdown Preview Enhanced: Open Mermaid Config</strong>  
  Editar la configuración de inicialización de `mermaid`.

- <strong> Markdown Preview Enhanced: Open Mathjax Config </strong>  
   Editar la configuración de inicialización de `MathJax`.

- <strong>Markdown Preview Enhanced: Image Helper</strong>  
  Para más información, consulta [este documento](image-helper.md).  
   El Ayudante de imágenes permite insertar URLs de imágenes rápidamente, pegar imágenes y subir imágenes con [imgur](https://imgur.com/) y [sm.ms](https://sm.ms/).  
  ![screen shot 2017-06-06 at 3 42 31 pm](https://user-images.githubusercontent.com/1908863/26850896-c43be8e2-4ace-11e7-802d-6a7b51bf3130.png)

- <strong>Markdown Preview Enhanced: Show Uploaded Images</strong>  
  Abrir `image_history.md`, que almacena la información de tus imágenes subidas.  
  Puedes modificar libremente el archivo `image_history.md`.

- <strong>Markdown Preview Enhanced: Run Code Chunk </strong>  
  <kbd>shift-enter</kbd>  
  Ejecutar un único [Code Chunk](code-chunk.md).

- <strong>Markdown Preview Enhanced: Run All Code Chunks </strong>  
  <kbd>ctrl-shift-enter</kbd>  
  Ejecutar todos los [Code Chunks](code-chunk.md).

- <strong>Markdown Preview Enhanced: Extend Parser</strong>  
  [Extender el Parser de Markdown](extend-parser.md).

_Vista previa_

**Haz clic derecho** en la vista previa para abrir el menú contextual:

![screen shot 2017-07-14 at 12 30 54 am](https://user-images.githubusercontent.com/1908863/28199502-b9ba39c6-682b-11e7-8bb9-89661100389e.png)

- <kbd>cmd-=</kbd> o <kbd>cmd-shift-=</kbd>.  
  Acercar la vista previa.

- <kbd>cmd--</kbd> o <kbd>cmd-shift-\_</kbd>.  
  Alejar la vista previa.

- <kbd>cmd-0</kbd>  
  Restablecer el zoom.

- <kbd>cmd-shift-s</kbd>  
  Desplazar el editor de Markdown para coincidir con la posición de la vista previa.

- <kbd>esc</kbd>  
  Mostrar/ocultar el TOC en la barra lateral.

## Atajos de teclado

| Atajo                                       | Funcionalidad                 |
| ------------------------------------------- | ----------------------------- |
| <kbd>ctrl-shift-m</kbd>                     | Activar vista previa          |
| <kbd>cmd-k v</kbd>                          | Abrir vista previa `Solo VSCode` |
| <kbd>ctrl-shift-s</kbd>                     | Sincronizar vista previa / fuente |
| <kbd>shift-enter</kbd>                      | Ejecutar Code Chunk           |
| <kbd>ctrl-shift-enter</kbd>                 | Ejecutar todos los Code Chunks |
| <kbd>cmd-=</kbd> o <kbd>cmd-shift-=</kbd>   | Acercar vista previa          |
| <kbd>cmd--</kbd> o <kbd>cmd-shift-\_</kbd>  | Alejar vista previa           |
| <kbd>cmd-0</kbd>                            | Restablecer zoom de vista previa |
| <kbd>esc</kbd>                              | Mostrar/ocultar TOC en barra lateral |

## Configuración del paquete

### Atom

Para abrir la configuración del paquete, presiona <kbd>cmd-shift-p</kbd> y elige `Settings View: Open`, luego haz clic en `Packages`.

Busca `markdown-preview-enhanced` en `Installed Packages`:  
![screen shot 2017-06-06 at 3 57 22 pm](https://user-images.githubusercontent.com/1908863/26851561-d6b1ca30-4ad0-11e7-96fd-6e436b5de45b.png)

Haz clic en el botón `Settings`:

![screen shot 2017-07-14 at 12 35 13 am](https://user-images.githubusercontent.com/1908863/28199574-50595dbc-682c-11e7-9d94-264e46387da8.png)

Ejecuta el comando `Preferences: Open User Settings`, luego busca `markdown-preview-enhanced`.
