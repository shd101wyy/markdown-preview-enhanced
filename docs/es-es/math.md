# Matemáticas

**Markdown Preview Enhanced** usa [KaTeX](https://github.com/Khan/KaTeX) o [MathJax](https://github.com/mathjax/MathJax) para renderizar expresiones matemáticas.

KaTeX es más rápido que MathJax, pero carece de muchas funciones que MathJax tiene. Puedes consultar las [funciones/símbolos admitidos por KaTeX](https://khan.github.io/KaTeX/function-support.html).

Por defecto:

- Las expresiones dentro de `$...$` o `\(...\)` se renderizarán en línea.
- Las expresiones dentro de `$$...$$` o `\[...\]` o <code>```math</code> se renderizarán como bloque.

![](https://cloud.githubusercontent.com/assets/1908863/14398210/0e408954-fda8-11e5-9eb4-562d7c0ca431.gif)

Puedes elegir tu método de renderizado matemático y cambiar los delimitadores matemáticos desde el [panel de configuración del paquete](usages.md?id=package-settings).

También puedes modificar la configuración de MathJax presionando <kbd>cmd-shift-p</kbd> y eligiendo el comando `Markdown Preview Enhanced: Open Mathjax config`.
