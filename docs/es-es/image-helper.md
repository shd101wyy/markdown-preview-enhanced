# Ayudante de imágenes

El ayudante de imágenes te ayuda a copiar y subir tus imágenes rápidamente.

## Usos

Para abrir el ayudante de imágenes, puedes ejecutar el comando `Markdown Preview Enhanced: Image Helper`, o hacer clic derecho en la vista previa y elegir la opción `Image Helper`.

![Screen Shot 2017-08-06 at 10.06.48 PM](https://i.loli.net/2017/08/07/5987d95bae68b.png)

## Subida rápida de imágenes (solo Atom)

Puedes subir una imagen rápidamente arrastrando tu archivo de imagen al editor de Markdown (no a la vista previa).
