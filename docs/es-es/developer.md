# Desarrollador

## Desarrollar este paquete para Atom

¡No dudes en enviar pull requests, reportar issues o solicitar nuevas funcionalidades!

Para modificar y desarrollar el paquete **markdown-preview-enhanced**, necesitas realizar la [instalación local](installation.md?id=install-from-github).

Después de instalar el paquete, sigue estos pasos:

- Abre la carpeta **markdown-preview-enhanced** en el **Editor Atom** desde **View->Developer->Open in Dev Mode...**
- Luego puedes modificar el código.
  Cada vez que actualices el código, presiona <kbd>cmd-shift-p</kbd> y elige `Window: Reload` para recargar el paquete y ver los cambios.

> Como la versión de Atom está escrita en TypeScript, se recomienda instalar `atom-typescript` para ayudar con el desarrollo del paquete.  
> Bueno... Desarrollé la versión de Atom usando vscode.

## Desarrollar este paquete para VS Code

Clona [vscode-markdown-preview-enhanced](https://github.com/shd101wyy/vscode-markdown-preview-enhanced), luego ejecuta `yarn install`, abre en **vscode** y comienza a depurar.
