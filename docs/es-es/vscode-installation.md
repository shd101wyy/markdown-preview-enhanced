# Instalación en VS Code

Hay varias formas de instalar este paquete.

## Instalar desde VS Code (Recomendado)

Abre el editor **vscode**, busca `markdown-preview-enhanced` en `Extensions`, luego haz clic en el botón `Install`.

También puedes instalar esta extensión en VS Code para la Web. Abre [VS Code for the web](https://vscode.dev/), busca `markdown-preview-enhanced` en `Extensions`, luego haz clic en el botón `Install`.

![screen shot 2017-07-14 at 12 23 32 am](https://user-images.githubusercontent.com/1908863/28199365-bb03a570-682a-11e7-8f65-d7d2b258d583.png)

## Instalar desde GitHub

- Descarga el archivo `markdown-preview-enhanced-*.vsix` desde [Releases](https://github.com/shd101wyy/vscode-markdown-preview-enhanced/releases).
- Abre **vscode**, ejecuta `Extension: Install from VSIX`, luego elige el archivo `*.vsix` que acabas de descargar.
