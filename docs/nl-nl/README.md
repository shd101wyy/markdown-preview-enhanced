# Markdown Preview Enhanced

**Markdown Preview Enhanced** is een **SUPERKRACHTIGE** markdown-extensie voor [**Visual Studio Code**](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced).
Het doel van dit project is om u een geweldige markdown-schrijfervaring te bieden.

Plaats [hier](https://github.com/shd101wyy/vscode-markdown-preview-enhanced/issues) een bericht als u nieuwe functies wilt aanvragen of bugs wilt melden ;)
