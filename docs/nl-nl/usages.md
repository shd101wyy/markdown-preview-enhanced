# Toepassingen

## Opdrachten

U kunt op <kbd>cmd-shift-p</kbd> drukken in de atom-editor om het <strong>opdrachtenpalet</strong> te openen.

> De <kbd>cmd</kbd>-toets is voor _Windows_ de <kbd>ctrl</kbd>-toets.

_Markdown-bron_

- <strong>Markdown Preview Enhanced: Toggle</strong>  
  <kbd>ctrl-shift-m</kbd>  
  Schakel de voorbeeldweergave van het Markdown-bestand in/uit.

- <strong>Markdown Preview Enhanced: Toggle Zen Mode </strong>  
  Schakel de afleidingsvrije schrijfmodus in/uit.

- <strong>Markdown Preview Enhanced: Customize Css</strong>  
  Pas de CSS van de voorbeeldpagina aan.  
  Hier is een korte [handleiding](customize-css.md).

- <strong>Markdown Preview Enhanced: Create Toc </strong>  
  Genereer een TOC (voorbeeldweergave moet actief zijn). [documentatie hier](toc.md).

- <strong>Markdown Preview Enhanced: Toggle Scroll Sync </strong>  
  Scroll-synchronisatie voor de voorbeeldweergave in-/uitschakelen.

- <strong>Markdown Preview Enhanced: Sync Source </strong>  
  <kbd>ctrl-shift-s</kbd>  
  Scroll de voorbeeldweergave naar de cursorpositie in de markdown-bron.

- <strong>Markdown Preview Enhanced: Toggle Live Update </strong>  
   Live bijwerken van de voorbeeldweergave in-/uitschakelen.  
   Als uitgeschakeld, wordt de voorbeeldweergave alleen weergegeven wanneer het bestand wordt opgeslagen.

- <strong>Markdown Preview Enhanced: Toggle Break On Single Newline </strong>  
  Regeleinde bij enkelvoudige nieuwe regel in-/uitschakelen.

- <strong>Markdown Preview Enhanced: Insert New Slide </strong>  
  Voeg een nieuwe dia in en ga naar de [presentatiemodus](presentation.md).

- <strong>Markdown Preview Enhanced: Insert Table </strong>  
  Voeg een markdown-tabel in.

- <strong>Markdown Preview Enhanced: Insert Page Break </strong>  
  Voeg een pagina-einde in.

- <strong> Markdown Preview Enhanced: Open Mermaid Config</strong>  
  Bewerk de `mermaid` initialisatieconfiguratie.

- <strong> Markdown Preview Enhanced: Open Mathjax Config </strong>  
   Bewerk de `MathJax` initialisatieconfiguratie.

- <strong>Markdown Preview Enhanced: Image Helper</strong>  
  Zie [deze documentatie](image-helper.md) voor meer informatie.  
   Afbeeldingshulp ondersteunt snelle invoeging van afbeeldings-URL's, afbeeldingen plakken en afbeeldingen uploaden via [imgur](https://imgur.com/) en [sm.ms](https://sm.ms/).  
  ![screen shot 2017-06-06 at 3 42 31 pm](https://user-images.githubusercontent.com/1908863/26850896-c43be8e2-4ace-11e7-802d-6a7b51bf3130.png)

- <strong>Markdown Preview Enhanced: Show Uploaded Images</strong>  
  Open `image_history.md` dat uw geüploade afbeeldingsinformatie opslaat.  
  U kunt het `image_history.md`-bestand vrij wijzigen.

- <strong>Markdown Preview Enhanced: Run Code Chunk </strong>  
  <kbd>shift-enter</kbd>  
  Voer een enkel [Code Chunk](code-chunk.md) uit.

- <strong>Markdown Preview Enhanced: Run All Code Chunks </strong>  
  <kbd>ctrl-shift-enter</kbd>  
  Voer alle [Code Chunks](code-chunk.md) uit.

- <strong>Markdown Preview Enhanced: Extend Parser</strong>  
  [Markdown-parser uitbreiden](extend-parser.md).

_Voorbeeldweergave_

**Klik met de rechtermuisknop** op de voorbeeldweergave om het contextmenu te openen:

![screen shot 2017-07-14 at 12 30 54 am](https://user-images.githubusercontent.com/1908863/28199502-b9ba39c6-682b-11e7-8bb9-89661100389e.png)

- <kbd>cmd-=</kbd> of <kbd>cmd-shift-=</kbd>.  
  Zoom in op de voorbeeldweergave.

- <kbd>cmd--</kbd> of <kbd>cmd-shift-\_</kbd>.  
  Zoom uit op de voorbeeldweergave.

- <kbd>cmd-0</kbd>  
  Zoom resetten.

- <kbd>cmd-shift-s</kbd>  
  Scroll de markdown-editor naar de positie van de voorbeeldweergave.

- <kbd>esc</kbd>  
  Zijbalk-TOC in-/uitschakelen.

## Sneltoetsen

| Sneltoets                                   | Functionaliteit                |
| ------------------------------------------- | ------------------------------ |
| <kbd>ctrl-shift-m</kbd>                     | Voorbeeldweergave in-/uitschakelen |
| <kbd>cmd-k v</kbd>                          | Voorbeeldweergave openen `Alleen VSCode` |
| <kbd>ctrl-shift-s</kbd>                     | Voorbeeldweergave synchroniseren / Bron synchroniseren |
| <kbd>shift-enter</kbd>                      | Code Chunk uitvoeren           |
| <kbd>ctrl-shift-enter</kbd>                 | Alle Code Chunks uitvoeren     |
| <kbd>cmd-=</kbd> of <kbd>cmd-shift-=</kbd>  | Voorbeeldweergave inzoomen     |
| <kbd>cmd--</kbd> of <kbd>cmd-shift-\_</kbd> | Voorbeeldweergave uitzoomen    |
| <kbd>cmd-0</kbd>                            | Zoom van voorbeeldweergave resetten |
| <kbd>esc</kbd>                              | Zijbalk-TOC in-/uitschakelen   |

Om de pakketinstellingen te openen, drukt u op <kbd>cmd-shift-p</kbd> en kiest u `Settings View: Open`, klik vervolgens op `Packages`.
