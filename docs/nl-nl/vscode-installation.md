# Installatie op VS Code

Er zijn verschillende manieren om dit pakket te installeren.

## Installeren vanuit VS Code (Aanbevolen)

Open de **vscode**-editor, zoek naar `markdown-preview-enhanced` in `Extensions` en klik vervolgens op de knop `Install`.

U kunt deze extensie ook installeren op VS Code voor het web. Open gewoon [VS Code voor het web](https://vscode.dev/), zoek naar `markdown-preview-enhanced` in `Extensions` en klik op de knop `Install`.

![screen shot 2017-07-14 at 12 23 32 am](https://user-images.githubusercontent.com/1908863/28199365-bb03a570-682a-11e7-8f65-d7d2b258d583.png)

## Installeren vanuit GitHub

- Download het bestand `markdown-preview-enhanced-*.vsix` van [Releases](https://github.com/shd101wyy/vscode-markdown-preview-enhanced/releases).
- Open **vscode**, voer `Extension: Install from VSIX` uit en kies vervolgens het `*.vsix`-bestand dat u zojuist heeft gedownload.
