# Markdown Preview Enhanced

**Markdown Preview Enhanced** est une extension Markdown **TRÈS PUISSANTE** pour [**Visual Studio Code**](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced).
L'objectif de ce projet est de vous offrir une expérience d'écriture Markdown merveilleuse.

Postez [ici](https://github.com/shd101wyy/vscode-markdown-preview-enhanced/issues) si vous souhaitez de nouvelles fonctionnalités ou si vous voulez signaler des bogues ;)
