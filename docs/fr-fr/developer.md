# Développeur

## Développer ce paquet pour Atom

N'hésitez pas à soumettre des pull requests, signaler des problèmes, ou demander de nouvelles fonctionnalités !

Pour modifier et développer le paquet **markdown-preview-enhanced**, vous devez effectuer une [installation locale](installation.md?id=install-from-github).

Après avoir installé le paquet, suivez ces étapes :

- Ouvrez le dossier **markdown-preview-enhanced** dans **Atom Editor** via **View->Developer->Open in Dev Mode...**
- Vous pouvez alors modifier le code.
  Après chaque mise à jour du code, vous devez appuyer sur <kbd>cmd-shift-p</kbd> puis choisir `Window: Reload` pour recharger le paquet et voir les modifications.

> Comme la version Atom est écrite en TypeScript, il est recommandé d'installer `atom-typescript` pour aider au développement du paquet.  
> Ceci dit... J'ai développé la version Atom en utilisant vscode.

## Développer ce paquet pour VS Code

Clonez [vscode-markdown-preview-enhanced](https://github.com/shd101wyy/vscode-markdown-preview-enhanced), puis `yarn install`, ouvrez dans **vscode** et commencez le débogage.
