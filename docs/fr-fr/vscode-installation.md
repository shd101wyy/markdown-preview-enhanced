# Installation sur VS Code

Il existe plusieurs façons d'installer ce paquet.

## Installer depuis VS Code (Recommandé)

Ouvrez l'éditeur **vscode**, puis recherchez `markdown-preview-enhanced` dans `Extensions`, puis cliquez sur le bouton `Install`.

Vous pouvez également installer cette extension sur VS Code pour le Web. Il suffit d'ouvrir [VS Code pour le web](https://vscode.dev/), puis de rechercher `markdown-preview-enhanced` dans `Extensions`, puis de cliquer sur le bouton `Install`.

![screen shot 2017-07-14 at 12 23 32 am](https://user-images.githubusercontent.com/1908863/28199365-bb03a570-682a-11e7-8f65-d7d2b258d583.png)

## Installer depuis GitHub

- Téléchargez le fichier `markdown-preview-enhanced-*.vsix` depuis les [Releases](https://github.com/shd101wyy/vscode-markdown-preview-enhanced/releases).
- Ouvrez **vscode**, exécutez `Extension: Install from VSIX`, puis choisissez le fichier `*.vsix` que vous venez de télécharger.
