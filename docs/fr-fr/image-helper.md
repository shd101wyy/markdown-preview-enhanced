# Aide aux images

L'aide aux images vous aide à copier et téléverser rapidement vos images.

## Utilisation

Pour ouvrir l'aide aux images, vous pouvez exécuter la commande `Markdown Preview Enhanced: Image Helper`, ou cliquer avec le bouton droit sur l'aperçu, puis choisir l'option `Image Helper`.

![Screen Shot 2017-08-06 at 10.06.48 PM](https://i.loli.net/2017/08/07/5987d95bae68b.png)

## Téléversement rapide d'images (Atom uniquement)

Vous pouvez téléverser rapidement une image en faisant simplement glisser votre fichier image vers votre éditeur Markdown (pas l'aperçu).
