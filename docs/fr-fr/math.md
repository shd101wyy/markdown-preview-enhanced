# Mathématiques

**Markdown Preview Enhanced** utilise [KaTeX](https://github.com/Khan/KaTeX) ou [MathJax](https://github.com/mathjax/MathJax) pour rendre les expressions mathématiques.

KaTeX est plus rapide que MathJax, mais il lui manque de nombreuses fonctionnalités que MathJax possède. Vous pouvez consulter les [fonctions/symboles pris en charge par KaTeX](https://khan.github.io/KaTeX/function-support.html).

Par défaut :

- Les expressions dans `$...$` ou `\(...\)` seront rendues en ligne.
- Les expressions dans `$$...$$` ou `\[...\]` ou <code>```math</code> seront rendues en bloc.

![](https://cloud.githubusercontent.com/assets/1908863/14398210/0e408954-fda8-11e5-9eb4-562d7c0ca431.gif)

Vous pouvez choisir votre méthode de rendu mathématique et modifier les délimiteurs mathématiques depuis le [panneau des paramètres du paquet](usages.md?id=package-settings).

Vous pouvez également modifier la configuration de MathJax en appuyant sur <kbd>cmd-shift-p</kbd> puis en choisissant la commande `Markdown Preview Enhanced: Open Mathjax config`.
